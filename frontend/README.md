# Oscar Legal Practitioners - Frontend

This is the frontend module for Oscar Legal Practitioners, a Virtual Legal Practice System.

## Architecture
Built with Vanilla HTML, CSS, and JavaScript as per academic requirements to demonstrate core web technologies.

- **index.html**: Public landing page.
- **assets/css/main.css**: Core design system and styles.
- **assets/js/main.js**: Interactivity and API communication.
- **assets/js/api.js**: (Planned) Centralized API client.

## Design Principles
- **Modern Aesthetics**: Using Inter and Outfit typography.
- **Glassmorphism**: Subtle transparency and blur effects.
- **Responsiveness**: Mobile-first design focusing on accessibility.
- **Premium Feel**: High-quality imagery and smooth transitions.
